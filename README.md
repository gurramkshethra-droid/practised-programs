## ▶️ How to Run

### 🟢 C Programs

1. Compile:

   ```
   gcc filename.c -o output
   ```
2. Run:

   ```
   ./output
   ```

---

### ☕ Java Programs

1. Compile:

   ```
   javac Main.java
   ```
2. Run:

   ```
   java Main
   ```
